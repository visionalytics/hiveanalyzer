# hiveanalyzer
Repository of public files of the HiveAnalyzer: a software supporting evaluation of honey bee (apis mellifera) brood trials according to OECD 75 guideline. By its high degree of automation (automated honeycomb alignment, automated cell detection, and automated cell classification), the HiveAnalyzer highly facilitates brood stage assessment.
